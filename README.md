# J2ME Telegram Client (Nokia Retro Project)

Легковесный Telegram-клиент для кнопочных телефонов с поддержкой Java (J2ME). Работает через промежуточный Python-сервер, обеспечивая доступ к чатам, сообщениям и фото на ретро-устройствах.

## Структура проекта

- `/client` — Исходный код Java-клиента (J2ME, MIDP 2.0).
- `/server` — Python-сервер на базе Quart и Telethon.
- `.github/workflows` — Автоматическая сборка .jar и .jad файлов через GitHub Actions.

## Как запустить

### 1. Сервер (Python)
Сервер выступает в роли прокси-шлюза между Telegram API и Java-клиентом. Рекомендуется запускать в **PyCharm**, **VS Code** или напрямую через терминал (CMD/PowerShell).
1. Перейдите в папку `server`.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Создайте файл `config.py` в папке `server`, скопировав содержимое из `config.py.example`.
4. Впишите ваши `API_ID` и `API_HASH` (их нужно получить на https://my.telegram.org) в созданный `config.py`.
5. Запустите: `python server.py`.

### 2. Создание туннеля (Serveo)
Чтобы телефон мог подключиться к вашему серверу через интернет, используйте SSH-туннель:
1. Откройте новый терминал.
2. Выполните команду:
   ```bash
   ssh -o ServerAliveInterval=60 -R 80:localhost:5000 serveo.net
   ```
3. Скопируйте полученную ссылку (например, `https://...serveo.net`). Она понадобится для настройки клиента.

### 3. Клиент (Java)
1. Соберите проект или скачайте готовый .jar из разделов Artifacts в GitHub Actions.
2. Установите на телефон.
3. При запуске введите URL вашего сервера. **Важно: ссылку в клиенте на Nokia нужно обязательно указывать в формате `http://`, а не `https://`.** Например: `http://vash-server.serveo.net`.

## Возможности
- Авторизация по номеру телефона.
- Просмотр списка чатов.
- Чтение и отправка сообщений.
- Отображение фотографий в чате.
- Поддержка кириллицы (UTF-8).

## Технологии
- **Java ME**: MIDP 2.0, CLDC 1.1.
- **Python 3.10+**: Quart, Telethon, Pillow.
- **GitHub Actions**: Автоматизация сборки JAR.
